I can help you work through your uploaded project documents in either a brief or detailed way.

**Ask document questions**
Examples: *"What are the functional requirements?"* or *"Who owns the API integration workstream?"*

**Ask follow-up questions**
Examples: *"Tell me more about that"*, *"What about section 3?"*, or *"Explain that in more detail"*

**Compare documents or versions**
Example: *"Compare the BRD and PRD requirements"* or *"How does v2 differ from v1?"*

**Summarize documents**
Example: *"Summarize the test plan"* or *"Give me a quick overview of the BRD"*

**Focus the answer**
You can ask for a *brief answer*, a *detailed explanation*, a *step-by-step walkthrough*, or a *high-level recap*.

**Upload documents**
Use the file upload button or type `/upload` to add documents such as PDF, DOCX, or Markdown files.

**Filter by document type**
Use the settings panel if you want to narrow results to a specific document type.
