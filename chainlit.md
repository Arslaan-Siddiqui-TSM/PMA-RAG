# PMA-RAG Chatbot

Welcome to the **Project Management Assistant**! I can answer questions about your project documents including PRDs, BRDs, use case documents, technical specs, test plans, and more.

## Getting Started

1. **Upload documents** using the file upload button (PDF, DOCX, or Markdown)
2. **Ask questions** about your uploaded documents
3. **Filter by document type** using the settings panel

I will always cite the source document and section for my answers, along with a confidence level.
