# PMA-RAG

Your AI-powered **Project Management Assistant** — ask questions, get cited answers, and navigate your project documents with ease.

## What Can I Do?

- **Answer questions** about your PRDs, BRDs, technical specs, test plans, use cases, and more
- **Cite sources** with document name, page/section, and a relevance score for every answer
- **Provide confidence levels** so you know how reliable each response is
- **Filter by document type** to narrow results to exactly what you need

## Getting Started

1. **Upload documents** — click the attachment button to upload PDFs, DOCX, or Markdown files
2. **Classify each file** — you'll be prompted to tag each upload (PRD, BRD, Technical Spec, etc.)
3. **Ask away** — type any question about your project documents and get an answer with sources

## Supported Document Types

| Type | Examples |
|------|----------|
| PRD | Product Requirements Documents |
| BRD | Business Requirements Documents |
| Technical Spec | Architecture docs, API specs |
| Test Plan | QA strategies, test cases |
| Use Case | User stories, workflow descriptions |
| Functional Spec | Feature specifications |
| Non-Functional Spec | Performance, security requirements |

## Tips

- Upload multiple files at once for batch ingestion
- Use the document type filter to scope your queries
- Re-uploading the same file will be detected and skipped automatically
